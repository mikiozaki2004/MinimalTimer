# 1. プロジェクト概要

- **プロジェクト名：** Minimalist Visual Timer (Desktop Edition)
- **コンセプト：** 究極のミニマリズムを追求した、デスクトップ常駐型の視覚的タイマー。
- **目的：** PC作業中に余計な情報を排除し、集中力（ゾーン）を維持するための美しいツールを提供する。

---

# 2. デザイン・UI要件（オールインワン・サークル）

「全ての要素が1つの円に収まり、状況に応じて変化する」ステルスUIを採用します。

## 2.1 画面構成

- **基本形状：** 完璧な真円（サークル）の中に全ての情報を内包する。

**メイン表示（常時）：**

| 配置 | 内容 |
|------|------|
| 中央上部 | 残り時間（または経過時間）を極細のフォントで大きく表示 |
| 中央下部 | 現在時刻を表示。メインタイマーのすぐ下に配置し、視認性を高めるためサイズを大きく設定 |
| 外周 | 進行状況を示すプログレスリング（1.5px〜3px の繊細なライン） |

**操作系（ホバー時のみ出現）：**

| 配置 | 内容 |
|------|------|
| サークル内部下部 | 開始/停止、リセット、モード切替、テーマ切替ボタン |
| サークル最上部 | ピン留め（最前面固定）ボタン |

## 2.2 ステルス・インタラクション

- **非操作時：** ボタン類を完全に非表示（Opacity 0）にし、数字とリングのみの「美しい時計」のような佇まいにする。
- **マウスホバー時：** インタラクティブな要素がスムーズに浮き上がる。

---

# 3. アニメーション要件（最優先事項）

「生きているような滑らかな動き」を徹底的に作り込み、プロダクトの質感を高めます。

- **動的なレイアウト・シフト：**
  マウスホバー時、メインタイマーと現在時刻がセットで上方向にスライドし、空いたスペースに操作ボタンが下からフワッと現れる連動アニメーション。

- **プログレスリングの連続性：**
  1秒ごとの更新をカクつかせず、アニメーションカーブを用いて常に滑らかにリングが動く演出。

- **テーマ・フェード：**
  テーマ切り替え時、背景の透過度・文字色・リング色がじわっと変化するクロスフェード処理。

- **マイクロ・インタラクション：**
  - ボタンクリック時のアイコン回転、スケールアップ/ダウンの視覚フィードバック。
  - モードラベルのゆっくりとした拍動（パルス）アニメーション。

---

# 4. 機能要件

## 4.1 計測モード

| モード | 説明 |
|--------|------|
| カウントダウン | 指定時間からゼロへ |
| カウントアップ | ゼロから開始するストップウォッチ |

## 4.2 ウィンドウ制御

- **最前面表示（Always on Top）：** 他のウィンドウに隠れないよう固定する機能を切り替え可能。
- **背景透過（グラスモーフィズム）：** 背景を磨りガラス風、または完全透過に設定可能。
- **自由移動：** ウィンドウの任意の場所をドラッグして配置可能。
