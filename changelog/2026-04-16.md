# 2026-04-16

### やったこと

#### ポモドーロ設定ウィンドウの不具合を修正
- `src-tauri/capabilities/default.json` の `windows` リストに `pomo` を追加
  - 未登録だったため `emitTo`・`hide` 等の Tauri API が全て無音失敗していた
  - これにより「開始ボタンが効かない」「✕/開始でウィンドウが閉じない」が解消
- `src-tauri/src/lib.rs` の `get_or_create_pomo()` に `set_always_on_top(true)` を追加
  - メインウィンドウが前面固定のため、pomo も前面固定しないと裏に隠れていた

#### ポモドーロ・タスクウィンドウのテーマ同期
- `src/pomo.css` / `src/task.css` に `body.light` / `body.sunset` テーマ CSS を追加
- `src/pomo.js` / `src/task.js` で起動時に localStorage からテーマを読んで適用
- `src/main.js` のテーマ変更ボタンで `theme-changed` イベントを pomo・task ウィンドウへ送信
  - ウィンドウを開いたままテーマを変えても即時反映されるようになった
